# Vue 3 Game (Rock, Paper, Scissors)



## Installation


1. Clone the project
   ```sh
   git clone https://github.com/jumahmohammadi/rock-paper-scissors-vue
   ```
2. go to rock-paper-scissors-vue folder/directory


3. run npm install command 
   ```sh
   npm install
   ```
   
4. run npm run dev command
   ```sh
      npm run dev
   ```
